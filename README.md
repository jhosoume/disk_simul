Em /built estão os códigos necessários para compilar o drive virtual.
Para isso, faça:
    (Na pasta built)
    >>> make clean
    Garante que é iniciada uma compilação limpa, em seguida:
    >>> make

É gerada um arquivo binário que pode ser inicializado com:
    >>> ./my_drive
